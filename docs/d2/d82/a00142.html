[BONUS] Override for the virtual method RevertMove in super class BasePiece

If the King os on king's row, you must demote it to a Man and continue the reverting on the move on the man Else, revert the piece like you would with any piece by calling RevertMove of the superclass

Be careful, when demoting you lose the reference to the current Cell and Board! You can always call the base function using:

base.FunctionName(FunctionParameters);
